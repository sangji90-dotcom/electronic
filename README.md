# ⚡ 날씨 데이터 기반 전력 소비량 예측 (Electronic Project)
> **기온, 습도 등 기상 요인 변화에 따른 전력 부하 등급 예측 모델링**

## 📅 프로젝트 개요
- **일정**: 2025.03 (팀 프로젝트 및 개인 재구현)
- **배경**: 기온 변화에 따른 전력 소비의 변동성을 머신러닝 모델로 예측하여 효율적인 에너지 관리를 목표로 함

## 🛠 사용 기술 및 환경
- **Language**: Python (Google Colab)
- **Library**: Pandas, NumPy, XGBoost, Scikit-learn, Matplotlib, Seaborn
- **Data**: 한국전력공사 공동주택 전력·기상 융합 데이터 (약 220만 건 로직 반영)

## 📋 핵심 수행 내용
1. **데이터 전처리 및 라벨링**
   - 전력 사용량을 계약 전력으로 나눈 **'전력부하비율'** 계산
   - 계산된 비율을 바탕으로 전력 부하 상태를 **3단계(낮음, 보통, 높음)**로 범주화(Labeling)
2. **모델링 및 성능 비교**
   - **XGBoost (최종 선정)**: 앙상블 학습을 통해 복잡한 비선형 데이터에서 가장 우수한 예측 성능 도출
   - **Random Forest**: 과적합 방지 및 변수 중요도 파악을 위한 비교 모델로 활용
3. **재구현 포인트**
   - 과거 팀 프로젝트에서 설계했던 로직을 Python 코드로 직접 재구현하여 모델 작동 원리 및 데이터 파이프라인 숙달

## 📊 분석 결과
- 기온과 전력 부하 사이의 강한 양의 상관관계 확인
- **XGBoost** 모델을 통해 전력 부하 등급 예측 정확도 확보
- 피처 중요도(Feature Importance) 분석을 통해 '기온'이 전력 소비에 미치는 영향력이 가장 큼을 증명
